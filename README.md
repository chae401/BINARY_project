# HotDesking :office:
cv를 이용한 자율 좌석 서비스


<br/>

## ✔️ Part     

휴학생들(3명)


+ 주민지 : 프론트


+ 김해리 : CV


+ 김채원 : 백엔드  

<br/>

## 📁 프로젝트 기능


+ 회원가입


+ CV를 활용한 로그인


+ 좌석의 상태('이용 중', '이용 가능', '자리 비움') 정보를 제공


+ 등록된 회원은 '이용 가능' 상태인 좌석을 선택한 후 자유롭게 업무공간을 사용할 수 있음
